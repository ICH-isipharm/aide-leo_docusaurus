# Raccourcis clavier

Les raccourcis-clavier permettent de gagner du temps dans **LEO**.  
Cette page regroupe les principaux raccourcis par catégories.

---
<Details summary="Raccourcis métier">
| Touche(s)           | Action                                                       |
| ------------------- | ------------------------------------------------------------ |
| **Ctrl B**          | Consulter la messagerie sécurisée Mailiz                     |
| **Ctrl D**          | Ajouter un document (numérisation, import, lien Internet)    |
| **Ctrl E**          | Imprimer des étiquettes produit                              |
| **Ctrl G**          | Interroger le stock d’un produit sur les sites du groupement |
| **Ctrl H**          | Lancer la recherche                                          |
| **Ctrl I**          | Requête Infos produits (PharmaML)                            |
| **Ctrl K**          | Ajuster rapidement le stock d’un produit                     |
| **Ctrl L**          | Voir les dernières ventes d’un produit                       |
| **Ctrl O**          | Consulter les relevés de prix d’un produit                   |
| **Ctrl P**          | Saisir un n° de ticket de promis                             |
| **Ctrl R**          | Créer une ligne de commande rapide                           |
| **Ctrl T**          | Ouverture exceptionnelle du tiroir-caisse                    |
| **Ctrl U**          | Changer d’utilisateur                                        |
| **Ctrl F1**         | Afficher la liste des raccourcis intégrés                    |
| **Ctrl F2**         | Sélectionner un client et afficher son compte                |
| **Ctrl Shift + T**  | Accéder aux missions de santé                                |
| **Ctrl ↑ / Ctrl ↓** | Déplacer une ligne produit                                   |
</Details>

<Details summary="Navigation / Présentation">
| Touche(s)           | Action                                                       |
| ------------------- | ------------------------------------------------------------ |
| **Tab**             | Aller au composant suivant                                   |
| **Shift + Tab**     | Revenir au composant précédent                               |
| **Échap**           | Fermer l’écran en cours sans valider / Masquer la barre de menu |
| **Alt**             | Afficher les raccourcis des mini-tuiles                      |
| **Alt ↑ / Alt ↓**   | Réduire / Afficher la zone de recherche                      |
| **Ctrl + ⊞ Win**    | Afficher / masquer la barre de menu                          |
| **Ctrl ← / Ctrl →** | Agrandir / Réduire la barre des boutons (droite)             |
| **Ctrl ²**          | Naviguer entre les écrans ouverts (touche exposant)          |
</Details>

<Details summary="Raccourcis standards Windows">
| Touche(s)               | Action                                     |
| ------------------------ | ------------------------------------------ |
| **Alt + lettre soulignée** | Activer le bouton d’action correspondant |
| **Ctrl A**               | Tout sélectionner / désélectionner         |
| **Ctrl C / V / X**       | Copier / Coller / Couper                   |
| **Ctrl F**               | Lancer une nouvelle recherche              |
| **Ctrl N**               | Créer un nouvel élément                    |
| **Ctrl S**               | Enregistrer                                |
| **⊞ Win + ↑ / ⊞ Win + ↓**| Agrandir / Réduire la fenêtr
</Details>
